# wavedef

A synthesizer library built on the [Web Audio API](https://developer.mozilla.org/en-US/docs/Web/API/Web_Audio_API).

## Installation

```sh
$ npm install wavedef
```

## Test

To run tests, do:

```
$ npm test
```
